# Обновление дизайна мельниц

## 🌾 Новый дизайн мельниц

Мельницы были полностью переработаны в соответствии с классическим голландским стилем.

### ✨ Основные изменения:

#### 1. **Каменное основание**
- Серый камень (#9e9e9e)
- Текстурированные слои для реалистичности
- Более широкое и устойчивое основание (3.0-3.2 радиус)

#### 2. **Красный деревянный корпус**
- Яркий красный цвет (#c62828) - характерный для голландских мельниц
- Восьмиугольная форма (8 сегментов)
- Коническое сужение снизу вверх (от 2.6 до 1.9)

#### 3. **Белые декоративные полосы**
- **Горизонтальные полосы**: 4 уровня белых ободков
- **Вертикальные балки**: 8 белых угловых балок по всей высоте
- Создают характерный полосатый узор

#### 4. **Решетчатые лопасти**
- **Деревянная рама**: Трапециевидная форма
- **Решетка**: Светло-желтая/кремовая (#f4e4c1)
  - 5 горизонтальных перекладин
  - 5 вертикальных перекладин
  - Создают аутентичный решетчатый узор
- **Внешняя окантовка**: Темно-коричневая (#6d4c2f)

#### 5. **Темно-коричневая крыша**
- Коническая форма
- Темный цвет (#3e2723)
- Расположена выше для лучших пропорций

#### 6. **Улучшенные детали**
- Окна с теплым свечением
- Деревянная дверь с золотой ручкой
- Большая центральная ступица лопастей
- Более медленное вращение (0.6 вместо 0.8) для реализма

### 📊 Технические характеристики:

```typescript
// Размеры
Основание: 3.0-3.2 радиус, высота 0.8
Корпус: 1.9-2.6 радиус, высота 6.8
Крыша: 2.1 радиус, высота 1.8
Лопасти: длина 4.8, ширина 0.5-0.35

// Цвета
Камень: #9e9e9e, #757575
Красный корпус: #c62828
Белые полосы: #f5f5f5, #eeeeee
Крыша: #3e2723
Рама лопастей: #8b6f47
Решетка: #f4e4c1
Окантовка: #6d4c2f
```

### 🎨 Визуальные улучшения:

1. **Аутентичность**: Точно соответствует классическим голландским мельницам
2. **Детализация**: Решетчатые лопасти вместо сплошных
3. **Цветовая схема**: Красный + белый + серый камень
4. **Пропорции**: Более реалистичные размеры и соотношения
5. **Материалы**: Правильные значения roughness для дерева и камня
