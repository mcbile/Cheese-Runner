## Общая информация

**Cheese Runner** — 3D endless runner игра, где игрок управляет лабораторной мышью, собирающей буквы слова "KAASINO", уничтожающей препятствия и сражающейся с врагами.

### Базовые константы

| Параметр | Значение | Описание |
|----------|----------|----------|
| `LANE_WIDTH` | 2.2 | Ширина одной полосы |
| `JUMP_HEIGHT` | 2.5 | Высота прыжка |
| `JUMP_DURATION` | 0.6 сек | Длительность прыжка |
| `RUN_SPEED_BASE` | 18.0 м/с | Базовая скорость движения |
| `REMOVE_DISTANCE` | 20 | Объекты удаляются за спиной игрока |

## Рендеринг объектов

### Координатная система

- **X** — горизонтальная ось (лево-право)
- **Y** — вертикальная ось (вверх-вниз)
- **Z** — ось глубины (отрицательные значения — впереди игрока)

Игрок всегда находится в позиции **Z = 0**. Объекты спавнятся с отрицательным Z и движутся к игроку (Z увеличивается).

### Система полос (Lanes)

Полосы центрированы относительно X = 0:

| Уровень | Кол-во полос | Индексы | X-координаты |
|---------|--------------|---------|--------------|
| 1 | 3 | -1, 0, 1 | -2.2, 0, 2.2 |
| 2-4 | 4 | -1, 0, 1, 2 | -2.2, 0, 2.2, 4.4 |
| 5 | 5 | -2, -1, 0, 1, 2 | -4.4, -2.2, 0, 2.2, 4.4 |

**Формула позиции:** `X = lane * LANE_WIDTH`

### Дистанции спавна

Объекты спавнятся на разных расстояниях от игрока:

| Тип объекта | Базовая позиция Z | При скорости 30 м/с |
|-------------|-------------------|---------------------|
| Босс | -90 | -108 |
| Коллекционные | -85 | -103 |
| Враги | -80 | -98 |

**Формула динамического спавна:**
```
spawnZ = baseZ - (speed - 18) × 1.5
```

### Правила спавна по типам

#### Вероятности спавна (за один цикл)

```
90% — попытка спавна объекта
├── 93.3% — препятствие/враг
│   ├── Мышеловки (1-3 шт):
│   │   ├── 55% — 1 мышеловка
│   │   ├── 30% — 2 мышеловки
│   │   └── 15% — 3 мышеловки
│   ├── 60% шанс сыра на мышеловке (Y=1.2)
│   └── Змея: каждые 15 мышеловок
└── 6.7% — высокий сыр (Y=3.15)

3% — Speed Boost
1.5% — Slow Motion
При 1 жизни — Heal (кулдаун 45 сек)
Буквы — по интервалу дистанции
```

#### Минимальный интервал между спавнами

```
minGap = 22 + (speed × 0.6)
```

При базовой скорости 18 м/с: `minGap = 22 + 10.8 = 32.8 единиц`

## Персонажи

### Игрок (Мышь)

Детализированный 3D персонаж лабораторной мыши с анимацией и физикой.

#### Физика движения

| Параметр | Значение | Описание |
|----------|----------|----------|
| Stiffness | 100 | Жёсткость пружины |
| Damping | 24 | Демпфирование |
| Gravity | 50 | Сила гравитации |
| Jump Force | 16 | Сила прыжка |
| Max Lateral Velocity | 12 | Макс. боковая скорость |

**Формула пружинной физики:**
```
acceleration = (stiffness × displacement) - (damping × velocity)
velocity += acceleration × delta
position += velocity × delta
```

#### Hitbox

| Параметр | Значение |
|----------|----------|
| Высота | 1.8 |
| Z-порог коллизии | 2.0 |
| X-порог коллизии | 0.9 |

#### Управление

**Клавиатура:**
- ←/→ — смена полосы
- ↑ — прыжок
- Пробел — стрельба
- V — активация FPS Camera
- M — Mute

**Тач:**
- Свайп вверх — прыжок
- Джойстик — смена полосы
- Кнопка — стрельба

#### Неуязвимость после урона

- Длительность: **1500 мс**
- Визуальный эффект: мигание персонажа

### Змея (Snake)

Диагонально движущийся враг типа "кобра".

| Параметр | Значение |
|----------|----------|
| HP | 1 |
| Награда | 2× ставка |
| Интервал смены полосы | 1.0 сек |
| Боковая скорость | 18% от дороги |
| Скорость движения | 85% от дороги |
| Hitbox (Y) | 0 — 0.6 |

**Поведение:**
1. Спавнится на крайней полосе (случайно лево/право)
2. Движется диагонально, отскакивает от краёв
3. Меняет направление каждую секунду

**Триггеры спавна:**
- Каждые 15 мышеловок
- После разблокировки Enemy Rush

### Кот (Cat)

Враг с прыжковой атакой.

| Параметр | Значение |
|----------|----------|
| HP | 2 |
| Награда | 3× ставка |
| Боковая скорость | 10% от дороги |
| Позиция прыжка | Z = -7.5 |
| Высота прыжка | 2.5 |
| Длительность прыжка | 0.4 сек |
| Hitbox (Y) | 0 — 1.0 |

**Фазы атаки:**
1. **Приближение** (Z < -10): Случайное уклонение влево-вправо
2. **Прицеливание** (Z = -10): Фиксирует соседнюю полосу игрока
3. **Прыжок** (Z = -7.5): Атакует полосу игрока

**Триггеры спавна:**
- Мгновенно после каждых 2 убитых змей
- Только 1 кот ИЛИ 1 сова на дороге одновременно

### Сова (Owl/Eagle)

Летающий враг с пикирующими атаками.

| Параметр | Значение |
|----------|----------|
| HP | 3 |
| Награда | 5× ставка |
| Высота полёта | 5.0 |
| Высота пике | 1.5 |
| Макс. пикирований | 3 |
| Интервал уклонения | 1.0 сек |
| Боковая скорость | 20% от дороги |
| Боковая скорость в пике | 30% от дороги |
| Hitbox (Y) | 1.0 — 2.5 |

**Фазы атаки:**
1. **Приближение** (Z < -50): Спуск с высоты, 90% скорости
2. **Досягаемость** (Z = -45): Игрок может стрелять
3. **Пикирование** (Z = -15): Атака + уклонение, 110% скорости
4. **Отступление**: Возврат на Z = -50, 120% скорости
5. **Повтор**: До 3 пикирований, затем улетает

**Множители скорости:**
| Фаза | Множитель |
|------|-----------|
| Приближение | 0.9× |
| Пикирование | 1.1× |
| Отступление | 1.2× |

**Триггеры спавна:**
- Мгновенно после каждого убитого кота
- Только 1 кот ИЛИ 1 сова на дороге одновременно

### Босс (Ученый)

Главный враг в конце каждого уровня.

#### Здоровье по уровням

| Уровень | HP | Награда |
|---------|----|---------|
| 1 | 20 | 24× ставка |
| 2 | 30 | 36× ставка |
| 3 | 40 | 48× ставка |
| 4 | 50 | 60× ставка |
| 5 | 60 | 72× ставка |

**Формула:** `HP = 20 + (level - 1) × 10`
**Награда:** `HP × 1.2 × betAmount`

#### Позиции

| Параметр | Z-координата |
|----------|--------------|
| Спавн | -90 (+ speed factor) |
| Нормальная | -25 |
| Отступление | -45 |

#### Атаки

**1. Шприцы:**
- Интервал: 1.15 сек (базовый)
- Ускорение: +15% на каждом этапе урона
- Награда за уничтожение: 1× ставка

**2. Рывок:**
- Интервал: 10-20 сек (случайный)
- Ширина: 1/2/3 полосы (зависит от уровня)
- Фазы: отступление → заряд → атака → возврат → удержание

#### Множители скорости

| Фаза | Множитель |
|------|-----------|
| Вход | 1.0× |
| Заряд (charge) | 3.0× |
| Отступление (retreat) | 1.1× |
| Возврат (return) | 1.35× |
| Боковое движение | 0.2× |

#### Этапы урона

| Этап | HP осталось | Эффект |
|------|-------------|--------|
| 0 | 100-67% | Нормальная скорость атаки |
| 1 | 67-34% | +15% скорость атаки |
| 2 | <34% | +30% скорость атаки |

#### Анимация смерти

1. **Отброс** (0.8 сек): Босс отлетает на Z = -30
2. **Падение** (1.0 сек): Падение на землю
3. **Приземление**: Портал спавнится

| Firewall | 10 сек | 2-полосные огненные выстрелы | Каждые 50 очков наград |

| Heal | мгновенно | +1 жизнь (до maxLives) | При 1 HP, кулдаун 45 сек |

### Расходники (покупаются в магазине)

| Предмет | Цена | Эффект | Длительность |
|---------|------|--------|--------------|
| Full Heal | очки | +1 жизнь (до maxLives) | мгновенно |

| Slow Motion | очки | Замедление до базовой скорости | 15 сек |

Расходники хранятся в инвентаре и активируются вручную во время игры.

### Immortality (НЕ РЕАЛИЗОВАНО)

**Immortality** — мёртвый код, механика заложена, но не работает:
- Код активации есть (Shift/Enter → `activateImmortality()`)
- Длительность в коде: 5 сек
- **НО:** `hasImmortality` всегда `false`, нет способа получить
- **В магазине отсутствует** — не продаётся

### Cheese Fever бонусы

Во время Cheese Fever при столкновении с врагами игрок получает сыр вместо урона:

| Враг | Сыр (шт) |

## Приложение: Формулы

### Позиция на полосе
```
X = lane × 2.2
```

### Динамический спавн
```
spawnZ = baseZ - max(0, speed - 18) × 1.5
```

### Минимальный интервал спавна
```
gap = 22 + speed × 0.6
```

### HP босса
```
bossHP = 20 + (level - 1) × 10
```

### Награда босса
```
reward = bossHP × 1.2 × betAmount
```

### Интервал букв
```
interval = 150 × 1.5^(level - 1)
```

### Границы полос
```
minLane = -floor((laneCount - 1) / 2)
maxLane = ceil((laneCount - 1) / 2)
```
